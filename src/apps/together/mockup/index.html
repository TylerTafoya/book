  • perm_identityWhat is the name of your app?

    Jukebox

  • radioWhat collaborative activity can people do together using your app?

    People can add to, vote on, and create music playlists to listen to at parties and gatherings.

  • filter_dramaWhat is the ideal number of concurrent users?

    Jukebox supports any number of users!

  • reorderWhat skills do people need in order to do this activity?

    Just a good taste in music!

  • whatshotIs this activity for fun or for something practical?

    A little bit of both! Jukebox is designed to enhance your get-togethers, but is very useful for large events.

  • zoom_inHow can a person see what everyone else is doing?

    Everyone logged into a playlist can see real-time updates of the current song and party queue.

  • visibilityHow can a person see the most recent result of everyone's collaborative effort?

    Each song in the queue can be voted on which shifts the order of music. These changes are updated on each client's screen.

  • gradeWhat can an admin see (i.e., God's mode)?

    The playlist owner can see everything the clients can, but can also see everyone logged into the playlist. Administrators can also invite and block users, as well as manage playlist settings.
